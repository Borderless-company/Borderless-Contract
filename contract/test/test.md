# Test Documentation

## ServiceType Enum

The `ServiceType` enum is defined as follows:

```typescript
enum ServiceType {
  NON,           // 0
  AOI,           // 1
  GOVERNANCE,    // 2
  LETS_EXE,      // 3
  LETS_NON_EXE,  // 4
  LETS_SALE,     // 5
  TREASURY       // 6
}
```

This enum is used to identify different types of services in the system. Each value corresponds to a specific service type, starting from 0.
